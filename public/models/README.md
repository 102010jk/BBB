# Custom orbital-station model (optional)

Drop a single GLB file here named **`station.glb`** and the simulator's orbital
laser cinematic will use it instead of the built-in procedural station.

- Format: **glTF binary (`.glb`)**, named exactly `station.glb`
- Use only assets you are licensed to use — **CC0 / public-domain** is safest for
  a school project (e.g. Quaternius "Ultimate Space Kit", NASA 3D Resources,
  Poly Pizza CC0 models).
- The model is auto-scaled and re-centred, and oriented to face the planet.
- If no file is present here, the hand-built station is used automatically.
